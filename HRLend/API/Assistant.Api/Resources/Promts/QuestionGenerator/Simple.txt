Придумай {2} разных вопроса на тему: {программирование}.
Ответ возвращая строго в json формате не добавляя лишних фраз.
Пример json
{
"questions":[
{
"text": "Как дела?",

"type": "radiobutton",
{
"options":[
{
"is_true": true,
"text": "Отлично!"
},
{
"is_true": false,
"text": "Плохо!"
}
]
}
]
}
здесь type - это тип вопроса он обязателен и может быть двух типов: radiobutton - один правильный вопрос,
checkbox - несколько правильных вопросов), options - это варианты ответов, он то же обязателен,
text - это сам вопрос, он то же обязателен.
