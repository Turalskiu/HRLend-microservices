Имеется json структура вопроса
{question}
здесь type - это тип вопроса, его не надо менять, options - это варианты ответов,
text - это сам вопрос.
Тебе нужно максимально возможно перефразировать вопрос и ответы на данный вопрос не меняя смысл вопроса.
Если ответ не правильный (is_true равен false) то можешь менять его полностью.
Ответ строго возвращай в виде json, не надо добавлять лишних фраз.
